====================
``1-my_list`` module
====================

``print_sorted`` will print a sorted list

::

Importing function and class from module
>>> My_list = __import__("1-my_list").My_list

Now use it
>>> My_list([1, 4, 2, 3, 5])
[1, 4, 2, 3, 5]

>>> My_list([1, 4, 2, 3, 5]).print_sorted()
[1, 2, 3, 4, 5]

>>> My_list([1, 4, 2, 3, 5])
[1, 4, 2, 3, 5]
